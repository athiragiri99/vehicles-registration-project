**Vehicle Registration Management System**:
A web-based vehicle registration management application built using Flask and SQLite.
The system simulates vehicle registration workflows with features for adding, updating, searching, and managing registration records.

**Features:**
Add new vehicle records
Edit existing records
Delete vehicles
Search by owner name or plate number
Automatic status update:
Active
Pending
Expired (based on expiry date)
Texas-style plate validation:
ABC1234
ABC-1234

**Technologies Used:**
Python
Flask
SQLite
HTML/CSS

**Installation & Setup:**
Clone the repository-
git clone https://github.com/your-username/vehicle-registration-system.git
Navigate to project folder-
cd vehicle-registration-system
Install dependencies-
pip install flask
Run the application-
python app.py
Open in browser-
http://127.0.0.1:5000/
