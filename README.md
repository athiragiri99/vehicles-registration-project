**Vehicle Registration Management System**
A web-based vehicle registration management application built using Flask and SQLite.
The system simulates vehicle registration workflows with features for adding, updating, searching, and managing registration records.

**Features**
Add vehicle registration records
Edit and update vehicle details
Delete records
Search vehicles by owner or license plate
Track registration status:
Active
Expired
Pending
Dashboard with registration statistics
Date formatting in MM/DD/YYYY format

**Technologies Used**
Python
Flask
SQLite
HTML
CSS
